# Learning Python

A small collection of beginner-friendly Python projects created while learning the basics of programming, logic, user input, loops, functions, and simple packages.

This repository is mainly used for practising Python fundamentals through small command-line projects.

## Projects Included

### Dice Rolling Game

A simple dice rolling game that generates random dice values.

**Concepts practised:**

- Random numbers
- User input
- Loops
- Basic game logic

---

### Number Guessing Game

A command-line game where the user tries to guess a randomly generated number.

**Concepts practised:**

- Conditional statements
- Loops
- Random numbers
- Input validation

---

### QR Code Generator

A Python script that generates a QR code from user-provided data.

**Concepts practised:**

- Python packages
- File output
- Functions
- QR code generation

---

### Rock Paper Scissors

A command-line version of the classic Rock, Paper, Scissors game.

**Concepts practised:**

- Conditional logic
- Random choice
- User input
- Game flow
